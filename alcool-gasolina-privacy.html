Política de Privacidade — Vai de Álcool ou Gasolina?

A sua privacidade é importante para nós. O aplicativo Vai de Álcool ou Gasolina? foi desenvolvido com o objetivo de fornecer uma ferramenta simples para ajudar o usuário a decidir qual combustível compensa mais, sem coletar, armazenar ou compartilhar quaisquer dados pessoais dos usuários.

1. Coleta de Dados

O aplicativo não coleta nenhuma informação pessoal. Nenhum dado como nome, e-mail, localização, endereço IP ou qualquer outro identificador é solicitado ou armazenado.

2. Uso de Dados

As informações inseridas no aplicativo (preço do álcool e preço da gasolina) são processadas localmente no dispositivo, exclusivamente para realizar o cálculo da melhor opção de abastecimento. Nenhum dado é transmitido ou salvo.

3. Compartilhamento de Dados

O aplicativo não compartilha dados com terceiros, não possui integração com serviços externos, nem utiliza cookies ou outras tecnologias de rastreamento.

4. Segurança

Como o aplicativo não coleta nem armazena informações, não há risco de vazamento ou uso indevido de dados pessoais.

5. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Qualquer alteração será informada através da loja de aplicativos.
